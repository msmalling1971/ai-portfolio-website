# Matt Smalling Portfolio - Version 1

Static portfolio website for Matt Smalling, focused on infrastructure modernization, operations leadership, platform engineering, and AI infrastructure.

## Run Locally

Open `index.html` in a browser, or serve the folder with any static file server.

## Structure

- `index.html` - single-page portfolio with all primary sections
- `src/styles.css` - responsive visual system and layout
- `src/main.js` - navigation, reveal motion, and active section state
- `assets/infrastructure-topology.svg` - custom operating model visual
- `assets/ai-lab-architecture.svg` - AI lab operating architecture diagram
- `assets/modernization-flow.svg` - modernization case study diagram
- `assets/ai-runtime-flow.svg` - local LLM runtime diagram
